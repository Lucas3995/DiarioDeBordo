    **2. Add relation management to ViewModel:**
    - `Relacoes`: `ObservableCollection<RelacaoItemViewModel>` — each item has `Id`, `NomeTipo`, `TituloDestino`, `IsInversa`
    - `MostrandoFormularioRelacao`: bool toggle
    - `ConteudoAlvoSelecionado`: selected content for new relation
    - `SugestoesConteudo`: populated via query searching acervo (exclude current content and already-related), paginated

    - `NomeInversoNovoTipo`: text for inverse name when creating new type

    - `[RelayCommand] CancelarFormularioRelacao()` → hides form, clears inputs

    - `[RelayCommand] RemoverRelacaoAsync(Guid relacaoId)`:
      1. Show confirmation dialog (UI-SPEC destructive confirmation: "A relação será removida de ambos os conteúdos")
      2. If confirmed → send `RemoverRelacaoCommand` via MediatR
      3. If success → remove from `Relacoes` list
