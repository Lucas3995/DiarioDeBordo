    **2. Add session management to ViewModel:**
    - `Sessoes`: `ObservableCollection<SessaoItemViewModel>` — each has `Id`, `Titulo`, `CriadoEm`, `Classificacao`, `Nota`, `Anotacoes`, `IsGostei`, `IsNaoGostei`
    - Progress calculation:
      - `ProgressoTexto`: computed from `Sessoes.Count` and `TotalEsperadoSessoes`:
        - If `TotalEsperadoSessoes.HasValue`: `string.Format(Strings.Progresso_ComTotal, Sessoes.Count, TotalEsperadoSessoes.Value, ProgressoPorcentagem)`
        - If no total but sessions exist: `string.Format(Strings.Progresso_SemTotal, Sessoes.Count)`
        - If no sessions: `Strings.Progresso_Vazio`
      - `ProgressoPorcentagem`: `(decimal)Sessoes.Count / TotalEsperadoSessoes.Value * 100` (clamped to 100)
    - Miniform fields: `SessaoTitulo`, `SessaoData` (defaults to `DateTimeOffset.Now`), `SessaoAnotacoes`, `SessaoNota`, `SessaoClassificacao` (with `SessaoIsGostei`/`SessaoIsNaoGostei` toggle logic same as main classificacao)
    - `MostrandoFormularioSessao`: bool toggle
    - `MostrandoDetalhesSessao`: bool toggle for progressive disclosure
    - `TextoBotaoDetalhesSessao`: "Mais detalhes" / "Menos detalhes"
    - `PodeRegistrarSessao`: `!string.IsNullOrWhiteSpace(SessaoTitulo)`

    - `[RelayCommand] MostrarFormularioSessao()` → shows form, focuses título field
    - `[RelayCommand] FecharFormularioSessao()` → hides form, clears inputs
    - `[RelayCommand] ToggleDetalhesSessao()` → toggles optional fields
    - `[RelayCommand] RegistrarSessaoAsync()`:
      1. Send `RegistrarSessaoCommand(UsuarioId, ConteudoId, SessaoTitulo, SessaoAnotacoes, SessaoNota, SessaoClassificacao, FormatoMidia.Nenhum)` via MediatR
      2. If success → add new session to `Sessoes` at index 0 (most recent first), clear form inputs (but keep form OPEN per D-20), reset SessaoData to today, update ProgressoTexto
      3. If failure → show error
    - `[RelayCommand] AbrirSessaoAsync(Guid sessaoId)`:
      - Opens the child content's modal via `_dialogService.MostrarConteudoDetalheAsync(sessaoId)`
      - On return, reload sessions list to reflect any edits made in child modal
