    Key implementation — `ObterDescendentesAsync` with BFS (per RESEARCH.md Pattern 2):
    ```csharp
    public async Task<IReadOnlyList<Guid>> ObterDescendentesAsync(
        Guid coletaneaId, Guid usuarioId, CancellationToken ct)
    {
        var visited = new HashSet<Guid>();
        var queue = new Queue<Guid>();
        queue.Enqueue(coletaneaId);

        while (queue.Count > 0)
        {
            var current = queue.Dequeue();
            if (!visited.Add(current)) continue;

            var children = await _context.ConteudoColetaneas
                .Where(cc => cc.ColetaneaId == current)
                .Join(_context.Conteudos,
                    cc => cc.ConteudoId, c => c.Id,
                    (cc, c) => new { c.Id, c.Papel, c.UsuarioId })
                .Where(x => x.Papel == PapelConteudo.Coletanea
                            && x.UsuarioId == usuarioId)
                .Select(x => x.Id)
                .ToListAsync(ct);

            foreach (var child in children)
                queue.Enqueue(child);
        }

        visited.Remove(coletaneaId); // exclude self
        return visited.ToList().AsReadOnly();
    }
    ```

    `ObterProximaPosicaoAsync`: query `MAX(Posicao)` from ConteudoColetaneas where ColetaneaId matches, return max+1 (or 1 if empty).
    
    `AdicionarItemAsync`: `_context.ConteudoColetaneas.Add(item); await _context.SaveChangesAsync(ct);`
    
    `RemoverItemAsync`: find by composite key + join to Conteudos for usuarioId check, remove, save.
    
    `ObterItemAsync`: find by composite key + join to verify ownership.
    
    `AtualizarItemAsync`: update tracking entity, save.
    
    `ItemExisteNaColetaneaAsync`: `AnyAsync` on composite key.
    
    `AdicionarAsync`, `ObterPorIdAsync`, `AtualizarAsync`: delegate to Conteudos DbSet with `.Where(c => c.Papel == PapelConteudo.Coletanea && c.UsuarioId == usuarioId)` filter. Include Fontes and Imagens in ObterPorIdAsync.
