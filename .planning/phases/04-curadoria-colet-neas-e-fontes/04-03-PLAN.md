    **AdicionarItemNaColetaneaHandler logic:**
    1. `var coletanea = await _coletaneaRepo.ObterPorIdAsync(cmd.ColetaneaId, cmd.UsuarioId, ct)` — if null, return `Resultado.Failure<Unit>("Coletanea nao encontrada")`
    2. `var conteudo = await _conteudoRepo.ObterPorIdAsync(cmd.ConteudoId, cmd.UsuarioId, ct)` — if null, return `Resultado.Failure<Unit>("Conteudo nao encontrado")`
    3. Check self-reference: if `cmd.ColetaneaId == cmd.ConteudoId`, return `Resultado.Failure<Unit>("Nao e possivel adicionar uma coletanea a si mesma")`
    4. `var existe = await _coletaneaRepo.ItemExisteNaColetaneaAsync(cmd.ColetaneaId, cmd.ConteudoId, ct)` — if true, return `Resultado.Failure<Unit>("Item ja na coletanea")`
    5. **Cycle detection:** If `conteudo.Papel == PapelConteudo.Coletanea`:
       - `var descendentes = await _coletaneaRepo.ObterDescendentesAsync(cmd.ConteudoId, cmd.UsuarioId, ct)`
       - If `descendentes.Contains(cmd.ColetaneaId)`, return `Resultado.Failure<Unit>("Ciclo detectado na composicao")`
    6. `var posicao = await _coletaneaRepo.ObterProximaPosicaoAsync(cmd.ColetaneaId, ct)`
    7. Create `ConteudoColetanea` with `ColetaneaId`, `ConteudoId`, `Posicao=posicao`, `AnotacaoContextual=null`, `AdicionadoEm=DateTimeOffset.UtcNow`
    8. `await _coletaneaRepo.AdicionarItemAsync(item, ct)`
    9. Return `Resultado.Success(Unit.Value)`
